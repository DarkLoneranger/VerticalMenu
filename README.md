# VerticalMenu

一个可以垂直弹出的菜单，实现了条目点击事件，也可基于此Demo实现菜单水平弹出。
This is a menu that can pop out vertically and the subitem of menu can be set on ClickListener. Maybe you can modify it to HorizontalMenu based on this Demo.
